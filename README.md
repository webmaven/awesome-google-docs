# awesome-google-docs
A collection of Google Docs tips, tricks, and hidden features.

## Useful **unlisted Markdown-based shortcuts** in Google Docs that are not found in the menus:

  * **Code Block:** Type three backticks (\`\`\`) at the start of a line and press **Enter** (or **Space**). This creates a dedicated, unstyled code block area.
  * **Numbered List:** Type `1.` followed by a **Space**.
  * **Bulleted List:** Type `*` or `-` followed by a **Space**.
  * **Heading 1:** Type `#` followed by a **Space**.
  * **Heading 2:** Type `##` followed by a **Space**.
  * **Heading 3:** Type `###` followed by a **Space**.
  * **Horizontal Line:** Type three hyphens (`---`) and press **Enter**.
